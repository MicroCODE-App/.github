## A.I. HOT KEYS

Key commands to generate common phrases used to control a coding A.I. Agent.
These are defined in AutoHotKey (AKK) - C:\Users\Timothy McGuire\Documents\AutoHotkey\My_AI_HotKeys.ahk
Install AutoHotKey / Download from:
https://www.autohotkey.com/

ALT-SHIFT-C: CONCEPT/CHANGE/CORRECTION - Discuss ideas without generating code

ALT-SHIFT-D: DESIGN - Design detailed solution

ALT-SHIFT-P: PLAN - Create implementation plan

ALT-SHIFT-V: REVIEW - Review and validate the implementation plan

ALT-SHIFT-B: BRANCH - Create Git branches for required repos

ALT-SHIFT-I: IMPLEMENT - Execute the plan

ALT-SHIFT-L: LINT - Check and fix linting issues

ALT-SHIFT-T: TEST - Run tests

ALT-SHIFT-M: DOCUMENT - Document the solution

ALT-SHIFT-R: PULL REQUEST - Create PRs for all repos
